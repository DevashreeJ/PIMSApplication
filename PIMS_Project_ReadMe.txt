In this application, We enter the details of student fact template as specifed in the requirement. 
Each record of one distric can be exported in CSV format.
There are 2 views 
1) The application form view
2) The database view

I have used CSV helper for converting record to csv format.
I have used entity model for using the database from the server.

Special conditions :

The code has been written in Visual Studios 2017 but the server used is MSSQL 2014.
